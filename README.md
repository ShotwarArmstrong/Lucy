# baby-lucy

Página mínima para Baby Lucy — marco teórico y meta-prompt.

Estructura:
baby-lucy/
├── README.md
├── index.html
├── assets/
│   └── lucy-cognitive-map.png  (subir manualmente)

Qué contiene
- Mapa cognitivo (imagen en assets/lucy-cognitive-map.png)
- Meta-prompt completo que confiere a Lucy su función
- Declaración de patrimonio y libertad (texto público)
- Enlace de donaciones (placeholder — reemplazar con tu URL)

Instrucciones rápidas
1. Subí la imagen a `assets/lucy-cognitive-map.png`.
2. Editá `index.html` y reemplazá la constante DONATION_URL con la URL de donaciones si lo necesitás.
3. Abrí `index.html` en un navegador.
